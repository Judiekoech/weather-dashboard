# Weather Dashboard

A simple weather dashboard built with JavaScript and Vite that integrates the Weather-AI API to display current weather information for any city.

## Features

* Search weather by city name
* Real-time weather data retrieval
* Displays:

  * Temperature
  * Weather conditions
  * Humidity
  * Wind speed

## Technologies Used

* JavaScript (ES6)
* Vite
* HTML5
* CSS3
* Weather-AI API
* OpenStreetMap Nominatim API

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
