# 🌦️ Weather Dashboard (Weather-AI API Integration)

A clean and responsive weather application built with Vite and vanilla JavaScript.  
It integrates geolocation + Weather-AI API to display real-time weather data.

---

## 🚀 Features

- Search weather by city name
- Displays:
  - Temperature
  - Weather condition
  - Humidity
  - Wind speed
- Clean UI with real-time API integration

---

## 🛠️ Tech Stack

- Vanilla JavaScript (ES Modules)
- Vite (build tool)
- Weather-AI API
- OpenStreetMap Nominatim API
- HTML + CSS

---

## 📡 APIs Used

- Weather-AI API → https://weather-ai.co/docs  
- OpenStreetMap Geocoding API
